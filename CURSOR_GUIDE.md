# Cursorを使ったサイト編集ガイド

AIエディタ「Cursor」を使って、このサイトを編集し、変更を反映させる手順を説明します。

---

## 1. 最初の準備（リポジトリのインポート）

まだパソコンにプロジェクトのファイルがない場合の最初の手順です。

### Step 1: GitHubからクローン（ダウンロード）する
1. Cursorを起動します
2. スタート画面の **「Clone a Git Repository」**（または上部メニューの `Terminal` → `New Terminal` を開いて操作）をクリック
3. 以下のURLを貼り付けます：
   ```text
   https://github.com/nobu0917/nanami-secret-base.git
   ```
4. 保存先のフォルダを聞かれるので、パソコン内の好きな場所（「ドキュメント」など）を選択します
5. 「Would you like to open the cloned repository?」と聞かれたら **「Open」** をクリックします

これでCursorの中にサイトのファイル一式が表示されます！

---

## 2. サイトを編集する（CursorのAIを活用）

Cursorの最大の強みは、AIに指示を出すだけでコードを書いてくれることです。

### 方法A: チャットで全体的な変更を指示する（Cmd+L / Ctrl+L）
1. 画面右側（または左側）の **「Chat」** パネルを開きます（ショートカット: `Cmd+L` または `Ctrl+L`）
2. AIに日本語で指示を出します。
   > **例:** 「`index.html` のメインビジュアルの下に、新しく『ななみのプロフィール』セクションを追加して。デザインは既存のCSSに合わせてください。」
3. AIがコードを提案してくるので、良ければ **「Apply」** をクリックしてファイルに適用します。

### 方法B: ファイル内の特定の場所を直接編集する（Cmd+K / Ctrl+K）
1. 編集したいファイル（例：`index.html`）を開きます
2. 変更したい部分をマウスで選択（ハイライト）します
3. `Cmd+K`（Windowsは `Ctrl+K`）を押します
4. 出てきた入力欄に指示を書きます。
   > **例:** 「このボタンの色をピンクからブルーに変更して」
5. AIがコードを書き換えるので、良ければ **「Accept」**（または `Cmd+Enter`）を押します。

---

## 3. 編集結果をローカルで確認する

変更した内容がブラウザでどう見えるか確認します。

1. Cursorの画面下部にある **Terminal（ターミナル）** を開きます（`Ctrl+`` またはメニューから）
2. 以下のコマンドを入力して `Enter` を押します：
   ```bash
   python3 -m http.server 8000
   ```
3. ブラウザ（Chromeなど）を開き、URLバーに `http://localhost:8000` と入力します
4. サイトが表示されるので、変更が正しく反映されているか確認します
   - ※確認が終わったら、ターミナルで `Ctrl+C` を押すとサーバーが止まります。

---

## 4. 変更をネット上に反映させる（GitHubへプッシュ）

編集と確認が終わったら、その変更をGitHub（そしてNetlify）に反映させます。
Cursorの左側にある **「Source Control（ソース管理）」** アイコン（枝分かれしたマーク）を使います。

### Step 1: 変更を記録する（Commit）
1. 左側の **「Source Control」** アイコンをクリックします
2. 「Changes」の下に変更したファイルが並んでいます
3. 上部のメッセージ入力欄に、何を変更したかメモを書きます
   > **例:** 「トップページにプロフィールを追加」
4. **「Commit」** ボタンをクリックします（「ステージングしますか？」と聞かれたら「Yes/Always」を選択）

### Step 2: ネットにアップロードする（Push）
1. Commitが終わると、ボタンが **「Sync Changes」**（または **「Push」**）に変わります
2. そのボタンをクリックします
3. くるくる回るアイコンが止まれば完了です！

### 自動反映（Netlify）
GitHubにPushされると、Netlifyが自動的にそれを検知してサイトを更新します。数分待ってから、あなたのNetlifyのURL（例：`https://nanami-secret-base.netlify.app`）にアクセスすると、変更が反映されています。
